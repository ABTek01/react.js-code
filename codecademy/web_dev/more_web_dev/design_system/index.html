Design System

Color Schema

Light Blue

Light Blue color usage; we can apply the lightest color to the background of the website to not overwhelm the user with an overabundance of colors.

Blue

Blue color usage; applying the next shade of blue to the outline and lettering of the website, along with interactivity such as buttons, will keep the user engaged.

Dark Blue

Dark Blue color usage; using darker blues within the footer section or as footnote text coloring will help keep the user focused on the main content on of the website.

Fonts Style

Roboto-Mono

Font style uses the the boldest font weight along with and underline style demonstrating a title characteristic.

Raleway

The assigned style depicts a blue coloring along with a subheading feature.

Oswald

This third font style is being used with an overline showing that it will be used as a footer styled title.
